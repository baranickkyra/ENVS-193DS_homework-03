# ENVS-193DS_homework-03

Homework Three for ENVS193DS

## General Information

This repository contains all code, analysis, and visualizations for Homework 3 in ENVS 193DS: Data Science for Environmental Science. The main goal of this assignment is to explore the relationship between soil salinity and California pickleweed biomass at a coastal restoration site using Pearson’s and Spearman’s correlation tests, supported by exploratory data visualization and diagnostic checks. Additional components include personal data visualizations, an affective design project, and a statistical critique of a PFAS contamination study.

## Source and File Information

The data folder includes:

salinity-pickleweed.csv (field measurements of soil salinity (mS/cm) and pickleweed biomass (g) and personal_data.csv ( self‑collected data on phone pickups, sleep duration, and day type)

The code folder includes:

the full Quarto document containing all R code, exploratory data analysis, diagnostic plots, Pearson and Spearman correlation tests, statistical interpretations, personal data visualizations, affective visualization description, and PFAS study critique.
